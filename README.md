## English

### What is iSurround?

iSurround is a **lightweight fork** of [DisplayMagician](https://github.com/terrymacdonald/DisplayMagician), stripped down to only the core features: **saving/loading display configurations** (NVIDIA Surround, AMD Eyefinity, multi‑monitor) and **switching them via tray menu or global hotkeys**.

If you only need to quickly switch between different display layouts without all the extra complexity (game libraries, audio switching, shortcut management, wallpaper, FOV calculator, etc.), iSurround is for you.

### Features

- Save, rename, update, delete display profiles
- Assign global hotkeys to profiles (e.g. `Ctrl+Alt+F1`)
- Switch profiles via system tray menu
- Auto‑start with Windows, minimise to tray
- Single instance enforcement
- Profiles stored in `Documents\iSurround\Profiles` (fully portable)

### Logging

- Logging is **enabled but only records `Fatal` and `Error` levels** by default (no chatty logs).
- No user‑facing on/off switch (keeps UI minimal).
- To change log level, modify the source code.

### ⚠️ Important Note for AMD Users

> Development was done on **NVIDIA Surround**. The author **does not own any AMD GPU**, therefore **AMD Eyefinity functionality is untested**. Code for AMD is kept, but **no guarantee it works**. Testers with AMD cards are welcome to provide feedback.

## 中文

### 这是什么？

iSurround 是 [DisplayMagician](https://github.com/terrymacdonald/DisplayMagician) 的一个**精简版**，只保留了最核心的功能：**保存/加载显示配置（NVIDIA Surround / AMD Eyefinity / 多屏）** 以及 **通过托盘菜单或全局热键一键切换**。

如果你只需要快速切换不同的显示布局，不需要那些复杂功能（游戏库、音频切换、快捷方式、壁纸、FOV计算器……），那么 iSurround 就是为你准备的。

### 功能

- 保存、重命名、更新、删除显示配置
- 为每个配置设置全局热键（如 `Ctrl+Alt+F1`）
- 托盘菜单切换配置
- 开机自启动、最小化到托盘
- 单实例运行
- 配置文件存放在 `我的文档\iSurround\Profiles`（完全便携）

### 日志说明

- 日志**默认只记录 `Fatal` 和 `Error` 级别**，不会产生大量日志文件。
- 没有开放给用户手动开关（保持界面简洁）。
- 如需修改日志级别，请改源代码。

### ⚠️ AMD 显卡用户特别注意

> 开发环境为 **NVIDIA Surround**，作者**没有 AMD 显卡**，因此 **AMD Eyefinity 功能未经任何测试**。代码中虽然保留了 AMD 库，但**无法保证正常工作**。欢迎有 AMD 显卡的朋友测试反馈。
